Session Creation (Admin)

Admin clicks START SESSION button

A new live session is created in the database with:

type = "admin"

unique_id = <randomly generated UUID>

userurl = <auto-generated session link>

A video player is displayed on screen after session creation (with Play, Pause, Volume, Fullscreen etc.)

Session Access (Student/User)

When the userurl is opened in any browser/tab

The same video player loads automatically

The session is linked using the stored unique_id

Student also has full playback controls

Database Structure

Database: MongoDB Atlas (Cloud)
Collection: live_sessions

Field	Type	Description
_id	ObjectId	Auto generated primary key
type	String	admin or student
unique_id	String	UUID assigned to session
userurl	String	URL students can join with

Data is stored in MongoDB, not locally.

Example stored document:

{
  "_id": "67ab1e312fcaf34c8a81b294",
  "type": "admin",
